# ❓ Frequently Asked Questions

## General Questions

### What is PixelPerfect ImageResizer?
PixelPerfect ImageResizer is a modern, client-side image processing tool that allows you to resize, compress, and transform images directly in your browser without uploading them to any server.

### Is it free to use?
Yes, PixelPerfect ImageResizer is completely free and open-source under the MIT license.

### Do I need to create an account?
No account registration is required. You can use all features immediately without any signup process.

## Privacy & Security

### Are my images uploaded to a server?
No, all image processing happens locally in your browser. Your images never leave your device, ensuring complete privacy.

### Is my data collected or stored?
We don't collect any personal data or store your images. Only basic preferences (like theme settings) are saved locally in your browser.

### Can I use this offline?
Once loaded, the application works offline for basic functionality. However, you need an internet connection to load external resources like fonts and icons.

**Input formats:** JPEG, PNG, WebP, GIF

### What browsers are supported?

- **Edge:** 80+

### Does it work on mobile devices?
Yes, the application is fully responsive and optimized for mobile devices, tablets, and desktops.

## Usage Questions

### How do I resize an image?
1. Upload your image by dragging & dropping or clicking "Select File"
2. Choose your resize method (Custom, Presets, or Transform)
3. Set your desired dimensions and quality
4. Click "Process" and then "Download"

### What's the difference between the output formats?
- **JPEG:** Best for photos, smaller file sizes, no transparency
- **PNG:** Best for graphics with transparency, larger file sizes
- **WebP:** Modern format with excellent compression, smaller than JPEG
- **GIF:** For simple graphics and animations, limited colors

### How do I maintain image quality?
- Use quality settings between 80-95% for best results
- Choose PNG for images with text or sharp edges
- Use WebP for the best compression with minimal quality loss
- Avoid resizing images larger than their original size

### Can I batch process multiple images?
Currently, the application processes one image at a time. Batch processing may be added in future updates.

## Features

### What are preset sizes?
Preset sizes are common dimensions for popular use cases:
- **Social Media:** Instagram (1080×1080), Facebook Cover (1200×630)
- **Video:** Full HD (1920×1080), HD (1280×720)
- **Web:** Various standard web sizes

### Can I rotate images?
Yes, you can rotate images by 90° increments and flip them horizontally or vertically using the Transform tab.

### Does it support animated GIFs?
Yes, you can upload and process animated GIFs. The animation will be preserved in the output.

### Can I crop images?
Currently, the application focuses on resizing and basic transforms. Cropping functionality may be added in future updates.

## Performance

### Why is processing slow?
Processing speed depends on:
- **Image size:** Larger images take longer
- **Device performance:** Older devices may be slower
- **Quality settings:** Higher quality takes more time
- **Browser:** Some browsers are more optimized

### How can I improve performance?
- Use smaller images when possible
- Lower quality settings (70-80% is usually sufficient)
- Close other browser tabs to free memory
- Use a modern browser version

### Why does my browser freeze?
This can happen with very large images or on devices with limited memory. Try:
- Using smaller images
- Lowering quality settings
- Closing other applications
- Refreshing the page and trying again

## Troubleshooting

### My image won't upload
Check if:
- File format is supported (JPG, PNG, WebP, GIF)
- File size is under 50MB
- File isn't corrupted
- JavaScript is enabled in your browser

### The preview isn't showing
This might be due to:
- Browser security restrictions
- Corrupted image file
- Insufficient memory
- Try refreshing the page and uploading again

### Download isn't working
Ensure:
- Pop-up blocker isn't preventing downloads
- You have permission to save files
- Sufficient storage space is available
- Try a different browser

### Features aren't working
Make sure:
- Your browser is up to date
- JavaScript is enabled
- You're not in private/incognito mode (some features may be limited)
